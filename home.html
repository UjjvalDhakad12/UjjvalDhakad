HTML

HTML is the standard markup language used to create web pages.

CSS

CSS is used to style and layout web pages, including colors, fonts, and spacing.

JavaScript

JavaScript is a programming language used to create dynamic and interactive web content.

React

React is a JavaScript library for building user interfaces, maintained by Facebook.

Node.js

Node.js is a JavaScript runtime built on Chrome's V8 engine, allowing server-side scripting.

Next.js

Next.js is a React framework that provides a solution for server-side rendering and static site generation.
